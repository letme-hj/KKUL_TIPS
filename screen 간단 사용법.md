# screen 간단 사용법
[방법]<br>
1. 터미널에 `screen` 입력
2. 원하는 거 실행
3. `ctrl+a, d` 로 detach (`d` 입력하는 순간 알아서 빠져나와져야 함)

(screen 하나 돌리고 detach 해서 나왔을 때의 화면)<br>
